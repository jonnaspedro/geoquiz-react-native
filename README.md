#  GeoQuiz – Aplicativo Educacional de Geografia

O **GeoQuiz** é um aplicativo educacional desenvolvido como projeto da 2ª avaliação da disciplina de Programação 3.  
O aplicativo tem como objetivo auxiliar no aprendizado de Geografia por meio de quizzes interativos e dinâmicos.

##  Objetivo do Projeto

Proporcionar uma forma divertida e interativa de estudar Geografia, utilizando perguntas e respostas sobre temas como:
- Capitais
- Países
- Continentes
- Climas
- Curiosidades geográficas

##  Funcionalidades

- Quiz de Geografia com perguntas de múltipla escolha  
- Exibição da pontuação final  
- Navegação entre telas  
- Interface simples e intuitiva  
- Armazenamento da melhor pontuação do usuário  

##  Tecnologias Utilizadas

- React Native  
- Expo  
- JavaScript  
- Context API  
- AsyncStorage
